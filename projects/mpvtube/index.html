Dependencies

What You Need

🐍
Python
3.10 or newer
Core runtime. Any 3.10+ install works.
▶
mpv
on PATH (or set manually)
The actual media player. mpvTube is just a launcher for it.
⬇
yt-dlp
via requirements.txt
Queries YouTube and lists available formats per video.
📦
ffmpeg
optional on Linux, auto-install on Windows
Format merging. Windows users get a one-click install button in the GUI.
Setup

Install & Run

Linux

$ git clone https://github.com/synapse-dot/youtube_mpv_app.git copy
$ cd youtube_mpv_app copy
$ chmod +x install.sh && ./install.sh copy
$ python main.py TUI (default) copy
$ python main.py --gui GUI mode copy
$ python main.py --min Minimal TUI copy

Windows

Use the included batch helper. If conda is available it creates a youtube_mpv env; otherwise it falls back to a local .venv.

> cmd /c run.bat copy

If mpv isn't on your PATH, set it in the small field at the top of the app. The GUI's Test mpv button validates and saves the path for you.

Install ffmpeg (auto) button installs ffmpeg.exe to ~/.youtube_mpv/bin automatically.

Features

What It Can Do

TUI (Textual)
  • ↵ Interactive YouTube search within the terminal
  • ↕ Browse results with arrow keys
  • ◈ Choose video and audio quality separately
  • ⧖ Search history h
  • ◇ Bookmarks b
  • / Quick search jump /
  • ▶ mpv auto-launched, app waits and returns
GUI / General
  • ⎋ No browser, no ads, no recommendations rabbit hole
  • ✓ Test mpv button validates path and saves config
  • ⬇ One-click ffmpeg install on Windows
  • ◈ Three launch modes: TUI, GUI, minimal TUI
  • ⚙ Persistent config at ~/.config/mpvTube/
  • ↑ Format passed as --ytdl-format directly to mpv
Configuration

Where It Lives

Configuration is stored as JSON and persists between sessions:

~/.config/mpvTube/config.json

The config stores your mpv binary path, any custom flags, and other preferences set through the GUI or TUI. You can also edit it directly — it's plain JSON.
