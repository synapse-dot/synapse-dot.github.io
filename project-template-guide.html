Project Template Guide

A simple step-by-step guide for creating and publishing a project page in the Simulation & Modelling Society website.

What is this page for?

Every project in our society has its own web page. This page is like a “profile” for your project.

It contains links to your code, reports, notes, and final release — so anyone can understand what your project is and how it works.

What you will create

1. A project webpage

A simple HTML file that describes your simulation project.

2. A GitHub repository

Where your actual code lives and is updated.

3. Documentation files

Reports, math notes, and development logs (PDF or Markdown).

How the system works

Think of your project like a folder with two parts:

  • Code (GitHub): The actual simulation program
  • Website page: The explanation and documentation of that project

The project template connects everything together in one place.

Step-by-step: How to use the template

Step 1 — Copy the template

Go to templates/project-ex.html and copy it. Rename it to your project name (example: project-pendulum.html).

Step 2 — Edit the title

Change “Project Title” to your actual project name and write a short description.

Step 3 — Add your links

Replace placeholder links with:

  • GitHub repository
  • Release version (e.g. v1.0.0)
  • PDF reports
  • Dev logs (Markdown files)

Step 4 — Add your team

Add contributor cards for each member working on the project.

Important rules

  • Every project must have a GitHub repository
  • Every project must have at least one report (Simulation Report)
  • Every project must have a version number (example: v1.0.0)
  • Documentation must explain what the simulation does and what it shows

Why we do it this way

This system is designed so that every project is:

  • Easy to understand for new members
  • Easy to share with others
  • Properly documented like real scientific work

It also helps you learn how real engineering and research projects are structured.

HELP

Still confused?

If anything feels unclear, don’t worry — this system is meant to be learned gradually while building your first project.
